# 👋 Welcome New Team Member!
## How to Register for Hospital Management System

---

## 🎯 What You'll Get

Once registered and approved, you'll have access to:
- 🏥 **Customer Database** - View all hospital customers
- 📊 **Visit Reports** - Submit and track your visits
- 📈 **Performance Dashboard** - See your progress
- 👥 **Team Collaboration** - Work with your assigned team

---

## 📝 Registration Steps

### Step 1: Open the Registration Page
Your manager will give you this link:
```
register.html
```

### Step 2: Fill in Your Information

#### ✅ Required Fields (Must Fill):

**Full Name**
- Example: `Abebe Kebede`
- Use your real name as it appears on ID

**Username**
- Example: `abebe` or `abebe123`
- Only letters and numbers (no spaces!)
- This is what you'll use to login
- Choose something memorable
- Must be unique (nobody else can have it)

**Password**
- Example: `MySecure123`
- Minimum 6 characters
- Choose something secure but memorable
- You'll need this to login!

**Confirm Password**
- Type the exact same password again
- Must match your password

#### 📌 Optional Fields (Nice to Have):

**Phone Number**
- Example: `0911123456`
- So admin can contact you if needed

**Email**
- Example: `abebe@example.com`
- For future notifications (coming soon!)

**Preferred Team**
- Example: `Team 3`
- Which team do you want to join?
- Admin will consider this but may assign you differently

### Step 3: Submit Registration
- Click the big **"📝 Submit Registration"** button
- Wait for success message: ✅ Registration submitted successfully!
- You'll be redirected to login page

### Step 4: Wait for Approval
- Admin will review your registration
- Usually approved within 24 hours
- You'll be notified once approved (or contact admin)

### Step 5: Login!
- Once approved, go to `login.html`
- Enter your username and password
- Click "🔐 Login"
- Welcome to your dashboard! 🎉

## ❓ Common Questions

### Q: What if my username is already taken?
**A:** You'll see an error "Username already exists"  
**Solution:** Choose a different username and try again

### Q: Can I change my username later?
**A:** No, username is permanent once approved  
**Solution:** Choose carefully the first time!

### Q: Can I change my password later?
**A:** Not yet (feature coming soon)  
**Solution:** Contact admin if you need password reset

### Q: How long does approval take?
**A:** Usually same day, maximum 24 hours  
**Solution:** Contact your manager if longer than 24 hours

### Q: What if I make a mistake in my name?
**A:** Admin can fix it after approval  
**Solution:** Just mention it to admin

### Q: Can I register multiple times?
**A:** No, one account per person  
**Solution:** If you have issues, contact admin to delete old registration

### Q: Do I need to remember team number?
**A:** No, you'll see team name in the dropdown  
**Solution:** Just pick the one that sounds right

## 🚨 Troubleshooting

### Problem: "Username already exists"
**Cause:** Someone already has that username  
**Fix:** Try adding numbers (e.g., `abebe` → `abebe123`)

### Problem: "Passwords do not match"
**Cause:** Password and Confirm Password are different  
**Fix:** Type them again carefully, making sure they match exactly

### Problem: "Network error"
**Cause:** Internet connection or server issue  
**Fix:** Check internet, wait 1 minute, try again

### Problem: "Invalid username format"
**Cause:** Username has spaces or special characters  
**Fix:** Use only letters (a-z, A-Z) and numbers (0-9)

### Problem: Page won't load
**Cause:** File not found or wrong link  
**Fix:** Contact your manager for correct registration link

### Problem: Nothing happens after submit
**Cause:** Browser issue  
**Fix:** Open browser console (F12), look for errors, screenshot and send to IT
