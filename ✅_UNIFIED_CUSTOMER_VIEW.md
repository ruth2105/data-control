## 🔍 SMART MERGING RULES

### How Duplicates are Handled:

```
IF hospital exists in PDF:
  → UPDATE with latest visit data
  → Keep PDF equipment info
  → Update status if changed
  → Add "Updated" badge
  → Show last visit date

IF hospital is NEW:
  → ADD as new customer
  → Use visit report data
  → Add "NEW" badge
  → Show last visit date
  → Extract equipment from report
```

### Name Matching:

Hospitals are matched by name (case-insensitive):
- "Lotus MCH" = "lotus mch" = "LOTUS MCH" ✅
- "Lotus" ≠ "Lotus MCH" ❌
- "St. Mary" ≠ "St Mary's" ❌

**Tip**: Use consistent hospital names in forms!

## 🆘 TROUBLESHOOTING

### "I submitted a visit but don't see it in Customer Potential"
**Solution**:
1. Make sure Google Sheets is connected
2. Refresh the page (F5)
3. Click Customer Potential tab again
4. Wait 2 seconds for data to load
5. Check Visit Reports tab - if it's there, it will merge

### "Customer appears twice"
**Should not happen** - merging prevents duplicates
**If it does**:
- Check if names are slightly different
- Example: "St. Mary" vs "St Mary's"
- Use exact same name in future reports

### "Badge doesn't show"
**Check**:
- ✨ NEW badge = Hospital not in PDF (new customer)
- 🔄 UPDATED badge = Hospital in PDF + you visited
- No badge = From PDF only (not visited yet)

### "Statistics don't update"
**Solution**:
1. Close and reopen Customer Potential tab
2. It fetches fresh data each time
3. If still wrong, check browser console (F12)
