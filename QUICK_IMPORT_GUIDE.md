## 🔧 Troubleshooting

**Q: Hospital name doesn't match my database**
- Check alternate spellings (e.g., "MCH" vs "Maternal & Child Health")
- Some hospitals may be listed with abbreviations
- Create new record if genuinely new facility

**Q: Contact information conflicts with existing data**
- PDF data is from recent report - likely more current
- Verify with sales rep who last visited
- Keep both and flag for verification call

**Q: How to handle equipment brand names**
- Store exactly as shown in PDF
- Useful for competitive analysis
- Update when sales team confirms on-site
