## 🎯 使用场景 / Use Cases

### 场景1: 老板要查看所有机器客户
**步骤:**
1. 点击 "🏥 机器客户" 标签
2. 点击右上角 "📥 导出客户列表"
3. 打开Excel文件
4. 使用过滤器查看特定城市或型号

### 场景2: 业务员要汇报本周拜访
**步骤:**
1. 点击 "📋 拜访管理" 标签
2. 使用过滤器选择自己的名字
3. 点击 "📥 导出 Excel"
4. 打开Excel，切换到 "拜访报告" 工作表
5. 使用过滤器只看本周数据

### 场景3: 分析设备需求情况
**步骤:**
1. 点击 "📊 数据看板" 标签
2. 选择时间段（本月/本年等）
3. 点击 "🔬 设备分析"
4. 打开CSV文件
5. 查看各品牌设备的使用统计

### 场景4: 准备月度会议材料
**步骤:**
1. **客户列表**: 从机器客户和外部客户页面各导出一份
2. **拜访报告**: 从拜访管理页面导出完整报告
3. **时间段分析**: 从数据看板导出本月报告
4. **设备分析**: 从数据看板导出设备统计
5. **整理**: 将所有Excel/CSV文件整理到一个文件夹

## 💡 Excel使用技巧 / Excel Tips

### 1. 快速过滤客户
1. 打开导出的客户列表
2. 点击列标题的 ▼ 箭头
3. 选择要查看的城市或型号
4. 立即显示筛选结果

### 2. 按安装日期排序
1. 点击 "安装日期" 列的 ▼ 箭头
2. 选择 "升序排序" 或 "降序排序"
3. 最新/最早安装的客户在最上面

### 3. 查找特定医院
1. 按 Ctrl+F 打开查找对话框
2. 输入医院名称的部分文字
3. Excel会高亮显示匹配的单元格

### 4. 复制到PowerPoint
1. 在Excel中选择需要的行
2. Ctrl+C 复制
3. 切换到PowerPoint
4. Ctrl+V 粘贴为表格

## ⚠️ 注意事项 / Important Notes

### 1. 数据权限
- **业务员**: 可以导出所有客户，但拜访报告只能看到自己的
- **管理员**: 可以导出所有数据

### 2. 文件格式
- **Excel (.xlsx)**: 客户列表、完整拜访报告
- **CSV (.csv)**: 时间段报告、设备分析
- **CSV优势**: 文件小，加载快，适合大数据量

### 3. 导出时机
- **最佳时机**: 网络连接良好时
- **数据实时性**: 导出的是点击时的最新数据
- **建议**: 重要会议前提前导出，避免网络问题

### 4. 文件保存
- **自动下载**: 文件会自动下载到浏览器默认位置
- **文件名**: 包含日期，便于版本管理
- **建议**: 创建专门文件夹存放导出文件

## 🔧 故障排除 / Troubleshooting

### Q1: 点击导出按钮没反应？
**A:** 
- 检查网络连接
- 刷新页面重试
- 检查浏览器是否允许下载

### Q2: Excel文件打不开？
**A:**
- 确保安装了Microsoft Excel或兼容软件
- 尝试用Google Sheets打开
- 检查文件是否完整下载

### Q3: 导出的数据不完整？
**A:**
- 检查是否应用了过滤器
- 确认登录账号权限
- 重新导出一次

### Q4: CSV文件乱码？
**A:**
- 用Excel打开时选择 UTF-8 编码
- 或用记事本打开，另存为 UTF-8
- 使用Excel的"数据" → "从文本"功能导入
