## 💡 使用指南 / Usage Guide

### 如何编辑报告 / How to Edit a Report

**步骤 / Steps:**

1. **登录**
   - 使用您的账号登录

2. **找到报告**
   - 进入"📋 拜访管理"页面
   - 可以使用过滤器快速找到

3. **点击编辑**
   - 报告卡片右下角："✏️ 编辑"
   - 或打开详情后点击底部"✏️ 编辑"

4. **修改信息**
   - 表单自动展开并填充数据
   - 修改需要更改的字段
   - 所有字段都可以修改

5. **保存**
   - 点击"💾 更新报告"
   - 看到成功提示 ✅

### 如何查看特定业务员的报告 / How to View Specific Salesperson's Reports

**方法 / Method:**

1. **进入拜访管理**
   - 点击"📋 拜访管理"标签

2. **使用业务员过滤器**
   - 点击"业务员"下拉框
   - 选择业务员名称（例如：Ephrata Tekola）

3. **查看结果**
   - 只显示该业务员的报告
   - 可以再组合其他过滤器

4. **重置过滤器**
   - 选择"全部"查看所有报告

### 如何按日期查看报告 / How to View Reports by Date

**方法 / Method:**

1. **进入拜访管理**
   - 点击"📋 拜访管理"标签

2. **使用时间范围过滤器**
   - 点击"时间范围"下拉框
   - 选择：今日/本周/本月/本年/自定义

3. **查看结果**
   - 只显示该时间段的报告
