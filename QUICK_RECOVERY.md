# ⚡ Quick Recovery Guide

## If You Delete by Mistake - DO THIS:

---

## 🎯 Method 1: UNDO Button (Easiest - 5 seconds)

### After you delete something:

```
┌─────────────────────────────────────────────┐
│  Report deleted. Click UNDO to restore.    │
│                                             │
│  [↶ UNDO / 撤销]  [✕]                       │
└─────────────────────────────────────────────┘
```

**This notification appears at the bottom of your screen for 30 seconds.**

### Steps:
1. Delete something (by accident)
2. See the notification appear at bottom
3. Click **↶ UNDO / 撤销** button
4. ✅ DONE! Item restored instantly!

### Important:
- ⏰ You have **30 seconds** to click UNDO
- ✅ Works for reports
- ✅ Works for customers  
- ⚡ Instant recovery
- 🎯 Easiest method!

---

## 🎯 Method 2: Google Sheets Version History (100% Recovery - 2 minutes)

### For anything deleted more than 30 seconds ago:

### Steps:

**1. Open Google Sheets**
   - Go to your hospital data sheet

**2. Open Version History**
   ```
   File → Version history → See version history
   ```
   OR keyboard shortcut:
   - Windows: `Ctrl + Alt + Shift + H`
   - Mac: `Cmd + Option + Shift + H`

**3. Find Version Before Deletion**
   - Look at the timestamps on the right
   - Click on a version from BEFORE you deleted
   - You'll see your data highlighted

**4. Restore**
   - Click **"Restore this version"** (blue button at top)
   - Confirm

**5. Refresh Your App**
   - Go back to your hospital app
   - Press `F5` or `Ctrl+R` to refresh
   - ✅ Data is back!

### Recovery Window:
- ✅ **30 days** for free Google accounts
- ✅ **Unlimited** for Google Workspace

## 🎯 Method 3: Export Backups (Prevention)

### Create Regular Backups:

**From Your App:**
```
📋 Visit Reports → 📥 Export Excel → Save with date
```

**From Google Sheets:**
```
File → Download → Microsoft Excel (.xlsx)
```

### Recommended Schedule:
- 📅 **Daily**: If making many changes
- 📅 **Weekly**: Normal usage  
- 📅 **Before major changes**: Always!

### File Naming:
```
backup_2024-06-30.xlsx
backup_2024-07-07.xlsx
backup_2024-07-14.xlsx
```

## 💡 Best Practice

**Before Deleting:**
1. Double-check it's the right item
2. Remember: You can EDIT instead of DELETE
3. If unsure, just EDIT and clear the fields
4. Or archive by adding "DELETED" to the name

**After Deleting:**
1. Watch for UNDO notification (30 sec)
2. Click UNDO if it was a mistake
3. If missed, use Google Sheets History

**Weekly:**
1. Export a backup every Monday
2. Store in organized folder
3. Name with date: `backup_2024-06-30.xlsx`

## 📞 Need Help?

### Common Issues:

**"I don't see the UNDO notification"**
→ Check bottom of screen (may be hidden behind other windows)
→ If 30+ seconds passed, use Google Sheets History instead

**"UNDO button doesn't work"**
→ Check internet connection
→ Try Google Sheets History method instead

**"Can't find version in history"**
→ Look at the time stamps carefully
→ Click through several versions
→ Data will be highlighted in color

**"Restored wrong version"**
→ No problem! Just restore a different version
→ You can restore as many times as needed
