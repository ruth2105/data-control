## 📥 导出Excel报告 / Export Excel Report

**冻结窗格：**
- 标题行始终可见

- 点击列标题的 ▼ 箭头

**示例：只看Ephrata的Hot客户**
1. 业务员列 → ▼ → 只选"Ephrata"

## ✏️ 编辑和删除 / Edit and Delete

### 编辑客户信息（仅管理员）
1. 点击医院卡片打开详情
2. 点击底部"✏️ 编辑客户"
3. 修改信息
4. 点击"💾 保存修改"

### 编辑拜访报告
1. 进入"📋 拜访管理"
2. 找到要编辑的报告
3. 点击"✏️ 编辑"
4. 修改后点击"提交"

**注意：业务员只能编辑自己的报告！**

### 删除（仅管理员）
- 删除前会要求确认
- 删除后无法恢复
- 谨慎操作！

## ❓ 常见问题 / FAQ

### Q1: 忘记密码怎么办？
**A:** 联系管理员重置密码。
默认密码：业务员=1234，管理员=admin123

### Q2: 提交报告失败怎么办？
**A:** 检查网络连接。系统会自动保存到本地，网络恢复后会同步到云端。

### Q3: 为什么看不到某些客户？
**A:** 检查过滤条件（城市、型号）是否选择了"全部"。

### Q4: Excel导出没有数据怎么办？
**A:** 需要先登录并提交至少一份拜访报告。

### Q5: 手机上能用吗？
**A:** 完全可以！系统100%支持手机、平板、电脑。

### Q6: 能离线使用吗？
**A:** 可以！数据保存在本地浏览器，离线也能查看和填写。联网后会自动同步。

### Q7: 怎么找到某个特定医院？
**A:** 使用搜索框，输入医院名称的任意部分，实时搜索。

### Q8: Excel过滤后怎么恢复？
**A:** 点击列标题的 ▼ → "清除筛选"，或点击Excel菜单的"数据" → "清除"。
