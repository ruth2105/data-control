## 📐 布局尺寸 / Layout Dimensions

### 桌面 / Desktop
- **模态框宽度**: 960px
- **左右列宽**: 各480px (50%)
- **内边距**: 20px 22px
- **分隔线**: 1px实线，颜色#d1fae5

### 平板 / Tablet
- **模态框宽度**: 90%屏幕宽度
- **最大宽度**: 960px
- **布局**: 根据屏幕宽度自适应（≥768px两列，<768px单列）

### 手机 / Mobile
- **模态框宽度**: 100%屏幕宽度
- **上下布局**: 公司产品在上，医院信息在下
- **分隔线**: 2px实线，颜色#10b981
- **内边距**: 20px 22px

## 🔍 信息层级 / Information Hierarchy

### 优先级 1 - 最重要 / Priority 1 - Most Important
**桌面**: 左侧 - 公司产品（老板最关心的）
**手机**: 上方 - 公司产品（先看到的）

内容包括：
1. ① 设备型号和安装日期
2. ② 库存情况（万孚产品）
3. ③ 订单需求（采购计划）

### 优先级 2 - 重要 / Priority 2 - Important
**桌面**: 右侧 - 医院信息
**手机**: 下方 - 医院信息（滚动查看）

内容包括：
1. ① 基本信息（名称、城市）
2. ② 联系人（姓名、电话）
3. ③ 付款联系人
4. ④ 医生信息

## ✅ 设计优势 / Design Advantages

### 1. **信息分组清晰 / Clear Information Grouping**
- 公司相关信息（绿色）和医院信息（灰色）明确分开
- 一眼就能找到需要的信息

### 2. **视觉引导强 / Strong Visual Guidance**
- 绿色=我们的产品（公司视角）
- 灰色=客户信息（客户视角）
- 颜色对比鲜明，不会混淆

### 3. **响应式设计 / Responsive Design**
- 桌面：左右并排，信息对比更直观
- 手机：上下排列，优先显示公司产品
- 适应所有设备屏幕

### 4. **易于扫描 / Easy to Scan**
- 编号系统（①②③④）
- 图标辅助识别
- 清晰的标题和分隔线

### 5. **商业逻辑清晰 / Clear Business Logic**
- 左侧/上方：我们卖给他们什么
- 右侧/下方：他们是谁，怎么联系
- 符合销售思维模式

## 🚀 使用指南 / Usage Guide

### 如何查看医院信息 / How to View Hospital Info

1. **进入客户列表**
   - 点击顶部"🏥 机器客户"或"🔬 外部客户"标签

2. **选择医院**
   - 点击任意医院卡片

3. **查看模态框**
   - **桌面用户**: 左右两栏同时可见
   - **手机用户**: 上下滚动查看

4. **关注重点**
   - **老板/管理员**: 主要看左侧（公司产品）
   - **业务员**: 两侧都要看，左侧了解产品情况，右侧联系客户

5. **操作按钮**
   - **编辑**: 点击底部"✏️ 编辑客户"按钮
   - **删除**: 点击底部"🗑 删除客户"按钮（仅管理员）
   - **关闭**: 点击右上角"×"或模态框外部区域

## 🎯 总结 / Summary

1. **业务优先 / Business First**

2. **信息清晰 / Clear Information**
   - 颜色编码：绿色=产品，灰色=客户
   - 不会混淆，快速定位

3. **全设备支持 / All Devices**
   - 桌面：左右布局
   - 手机：上下布局
   - 完美适配

4. **易于维护 / Easy Maintenance**
   - 代码结构清晰
   - 样式统一管理
   - 未来扩展方便
