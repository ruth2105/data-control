# 👥 User Registration Guide - How Senior Sales Get Access

## 🎯 Quick Answer

**TWO OPTIONS for getting senior sales set up:**

### **Option 1: Admin Creates Accounts** ✅ (RECOMMENDED)
- **Admin** creates accounts for all sales people
- No self-registration needed
- Admin controls access from day 1

### **Option 2: Self-Registration** ✅ (Also Available)
- Sales people can register themselves
- They get access immediately
- Admin can manage them later

---

## 📋 Option 1: Admin Creates Accounts (RECOMMENDED)

### **Why This is Better:**
- ✅ Admin has full control
- ✅ Assigns correct teams from start
- ✅ Sets proper permissions
- ✅ No duplicate accounts
- ✅ Professional onboarding

### **How Admin Creates Sales Accounts:**

#### Step 1: Admin Opens User Management
```
1. Open: app/admin.html
2. Login as admin
3. Click: "👥 User Management" tab
```

#### Step 2: Create New User
```
1. Click: "➕ Add New User" button
2. Fill form:
   - Full Name: "Kalkidan Tesfaye"
   - Username: "kalkidan" (lowercase, no spaces)
   - Password: Create secure password
   - Team: Select "Kalkidan" from dropdown
   - Role: Select "Sales"
   - Phone: Optional
3. Click: "Create User"
```

#### Step 3: Give Credentials to Sales Person
```
Send them:
- Website URL: https://your-site.netlify.app
- Username: kalkidan
- Password: [the password you created]
- Page to use: /app/sales.html
```

#### Step 4: Sales Person Logs In
```
1. Opens: app/sales.html
2. Enters username and password
3. Starts working!
```

---

## 📋 Option 2: Self-Registration

### **When to Use:**
- Sales people need access quickly
- Admin is not available
- Temporary access needed

### **How Sales Person Registers:**

#### Step 1: Open Registration Page
```
Navigate to: app/register.html
```

#### Step 2: Fill Registration Form
```
Full Name: Kalkidan Tesfaye
Username: kalkidan (lowercase, no spaces)
Password: [choose secure password]
Confirm Password: [same password]
Phone: [optional]
```

#### Step 3: Submit
```
1. Click: "📝 Submit Registration"
2. Account created immediately!
3. Redirect to login page
```

#### Step 4: Login
```
1. Opens: app/sales.html
2. Enters username and password
3. Starts working!
```

### **⚠️ Important Note:**
When self-registering:
- User is created with **default permissions**
- Admin should later:
  - Assign correct team
  - Verify role (Sales/Admin)
  - Update any missing information

---

## 🔐 User Roles & Permissions

### **Sales Role:**
- Can add/edit their own visit reports
- See their own customers
- View personal dashboard
- Announce purchases
- **Cannot** see other team's reports
- **Cannot** manage users

### **Admin Role:**
- Everything Sales can do, PLUS:
- See **all** team reports
- Boss dashboard with all teams
- User management
- Approve purchases
- Manage customer database
- Full system access

## 🔄 Changing Passwords

### **Sales Person Changes Their Password:**
```
1. Login to sales page
2. Click profile/settings icon
3. Click "Change Password"
4. Enter old password
5. Enter new password
6. Save
```

### **Admin Resets User Password:**
```
1. Login to admin page
2. Go to User Management
3. Find the user
4. Click "Reset Password"
5. Set new password
6. Give new password to user
```

## ❓ Common Questions

### **Q: Do senior sales need admin approval after registration?**
**A:** No! Registration is immediate. They can login and start working right away.

### **Q: Can sales register themselves or must admin create account?**
**A:** Both work! But **admin creating accounts is better** for control and proper setup.

### **Q: What if someone registers with wrong team?**
**A:** Admin can edit user details and change team assignment in User Management.

### **Q: Can one person have multiple accounts?**
**A:** Not recommended. Each person should have ONE account. Admin can change permissions/teams as needed.

### **Q: What if sales person forgets password?**
**A:** Admin can reset it in User Management, or they can use "Forgot Password" (if implemented).
